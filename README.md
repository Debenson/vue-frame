# Vue-frame

> A simple component Vue 2.0 for prototyping interfaces with iframes.

### Usage

This is a project template for [vue](https://github.com/vuejs).

``` bash
$ vue init browserify-simple my-project
$ cd my-project
$ npm install
$ npm run dev
```
